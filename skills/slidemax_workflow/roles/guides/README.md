# Role Guides

This directory stores focused reference guides for role documents.

## Purpose

- Keep role entry files short and stage-oriented.
- Move dense reference material into reusable guides.
- Provide stable extension points when a role gains more constraints.

## Guides

| Role | Guide | Purpose |
|------|-------|---------|
| Strategist | [strategist-confirmations.md](./strategist-confirmations.md) | Eight confirmations, style routing, typography, and icon policy |
| Strategist | [strategist-image-planning.md](./strategist-image-planning.md) | Image resource planning, `analyze_images`, and layout-fit rules |
| Strategist | [strategist-design-brief-contract.md](./strategist-design-brief-contract.md) | Required output structure and handoff contract |
| Image_Generator | [image-generator-prompt-pack.md](./image-generator-prompt-pack.md) | Prompt-file structure, asset typing, acquisition paths, and completion rules |

## Maintenance Rules

- Keep role entry files in `../` focused on stage execution and handoff.
- Keep provider-specific setup in `../../references/docs/`.
- Keep command detail in `../../references/docs/command_reference.md`.
